== Descripción ==

Este plugin agrega dos shortcodes:
- [fecha_actual] → Muestra la fecha completa en español (ej: "Lunes, 15 de abril de 2025")
- [anio_actual] → Muestra únicamente el año actual (ej: "2025")

== Instalación ==

1. Sube la carpeta `plugin-fecha-actual` al directorio `/wp-content/plugins/`
2. Activa el plugin desde el menú 'Plugins' en WordPress
3. Usa [fecha_actual] o [anio_actual] en cualquier entrada o página
